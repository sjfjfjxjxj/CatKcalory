# CatKcalory

## 주제: 고양이 필요 칼로리 계산 프로그램

 
## 요구사항
 - 고양이 이름 / 체중 / 나이 / 중성화 여부를 입력받는다.
 - 고양이 필요 칼로리 출력, 사료 양(건사료 평균치로)을 계산해준다.
   
   
## 동작화면
------------고양이 하루치 사료 양 계산하기-------------

0. 메뉴 
1. 고양이 정보 입력 
2. 고양이 정보 출력 
3. 고양이 필요 칼로리 출력 -> 사료 양 계산 
3. 프로그램 종료
